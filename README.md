# Superstore Sales Dashboard – Elevvo Internship Task 1

## Overview
This project analyzes the Superstore dataset using Microsoft Excel to uncover key business insights.

## Key Features
- Data cleaning and preparation
- Pivot table analysis
- Month-over-Month and Year-over-Year trends
- Interactive slicers
- Executive Dashboard

## Key Insights
- West and East regions contributed the highest profit.
- High discounts did not always result in higher profitability.
- Monthly trends highlighted seasonal sales patterns.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Data Visualization
